# Car Rental System – Java OOP Project

## Course
SWE 303 – Java OOP

## Description
This project implements a Car Rental Business system in Java. 
It allows clerks to manage cars, customers, and bookings, with 
automatic invoice generation and a simple GUI.

## Features
- Car registry management
- Clerk login
- Customer registration
- Car booking and cost calculation (with VAT)
- Invoice generation
- File handling for data persistence
- Swing GUI interface

## OOP Concepts Used
- Inheritance (Vehicle → Car)
- Abstract classes
- Polymorphism
- Encapsulation

## How to Run

2. Run `Main.java` in the `app` package
